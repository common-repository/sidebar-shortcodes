Allows shortcodes to be used in the sidebar text widget.

== Description ==

Once this plugin is activated, you'll be able to use the new "shortcodes"
in your sidebar text widget.  The shortcodes will work exactly the same
way as they do in your posts.

== Installation ==

Just download, unzip into your wp-content/plugins folder and activate.
